# alpha2048

2048ゲームのソルバーを目指すプロジェクト。

## 遊び方

```bash
go run ./cmd/play
```

### 操作方法

| キー | 方向 |
|------|------|
| w    | 上   |
| s    | 下   |
| a    | 左   |
| d    | 右   |
| q    | 終了 |

### 画面例

## 自動プレイ（ソルバー）

```bash
go run ./cmd/autoplay
```

### オプション

| フラグ | デフォルト | 説明 |
|--------|-----------|------|
| `-depth` | 3 | 探索深さ |
| `-delay` | 100 | 手の間隔（ミリ秒） |
| `-astar` | false | A*アルゴリズムを使用 |
| `-quiet` | false | 出力を抑制 |

### 例

```bash
# 深さ4で探索（より賢いが遅い）
go run ./cmd/autoplay -depth=4

# A*アルゴリズムを使用
go run ./cmd/autoplay -astar

# 高速実行（出力なし）
go run ./cmd/autoplay -delay=0 -quiet
```
