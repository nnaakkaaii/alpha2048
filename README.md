# alpha2048

2048ゲームのソルバーを目指すプロジェクト。

## 概要

2048は、4x4のグリッド上で同じ数字のタイルを合体させながら、2048のタイルを作ることを目指すパズルゲームです。このプロジェクトでは、ゲームの実装とAIソルバーの開発を行っています。

## ゲームルール

1. **初期状態**: 4x4のグリッドにランダムに2つのタイル（2または4）を配置
2. **操作**: 上下左右にスワイプして、タイルを移動・合体
3. **タイル生成**: 移動後、空きマスに新しいタイル（90%で2、10%で4）が出現
4. **終了条件**: どの方向にも移動できなくなったらゲームオーバー
